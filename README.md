# Scholarship-Like

<img src="https://user-images.githubusercontent.com/83286706/185306896-5953dcc8-d2f5-4c3d-9c1d-cf44a26dd6aa.png">


## 주제
맞춤 장학금 정보 제공 애플리케이션

## 개요
- ‘장학라이크’는 대학생을 대상으로 국가, 교내, 교외 장학금에 대한 통합적인 정보를 제공할 뿐만 아니라, 개인의 조건(학자금 지원 구간, 이수 학기, 학교 등)에 일치하는 장학금을 알려주는 ‘맞춤 장학금 정보 제공’ 애플리케이션입니다. 
- ‘장학라이크’만의 특징은 사용자의 조건을 실시간으로 바꿔보며 장학금을 비교할 수 있다는 점, 국가나 지역 지원금의 정보를 제공하며 범위가 넓다는 점입니다. 더불어, 캘린더와 알림 서비스 제공으로 사용자에게 장학금을 상기시키며 수혜율을 높일 수 있습니다.

## 과제 목적
‘장학라이크’의 목적은 많은 대학생들이 장학금에 할애하는 시간과 노력을 절약할 수 있도록 하고 대학 생활에 있어 더 나은 경제생활을 할 수 있도록 하는 것입니다.
